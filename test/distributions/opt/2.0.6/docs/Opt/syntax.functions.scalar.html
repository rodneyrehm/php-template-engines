3.6.28. scalar()

Referenceboolean scalar(mixed $value)
Versionssince 2.0-RC2

Returns true, if the specified value is a scalar, that is:

  • A string
  • An integer
  • A floating point number
  • A boolean value

False is returned for containers and PHP resources.
